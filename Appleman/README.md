Appleman (c) mjt, 2016

Released under MIT-license.
Requires Java.

Simple game made with libgdx.

Collect fresh apples, beware enemies.

PageUp/PageDown  changes screen size.


Run game:
  java -jar Appleman.jar


Compile to desktop:
  gradlew desktop:dist

Compile to web (dev):
  gradlew.bat html:superDev
  http://127.0.0.1:8080
